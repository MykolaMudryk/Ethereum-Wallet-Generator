**README.md**

Ethereum Wallet Generator

Цей скрипт є інструментом для генерації адрес гаманців Ethereum за допомогою приватних ключів. Приватні ключі зберігаються у файлі private_keys.txt, і для кожного приватного ключа генерується та записується в файл EVM_addresses.txt його відповідна адреса гаманця Ethereum у форматі 'private_key - EVM_adress'

### Використання на MAC

1. Встановлення залежностей:
   
   Запустіть наступну команду через термінал, щоб встановити необхідну бібліотеку `web3`:
 
   pip install -r requirements.txt

   Це забезпечить встановлення бібліотеки `web3` та інших залежностей, якщо їх ще немає.

2. Згенерувати адреси гаманців:

   Введіть приватні ключі у файл `private_keys.txt`, кожен приватний ключ на новому рядку.

3. Запуск генератора гаманців:

   Запустіть скрипт за допомогою команди:

   python wallet_generator.py

   Програма виведе повідомлення про успішну генерацію адрес та збереження у файл. Адреси гаманців з приватними ключами будуть збережені у файлі `EVM_addresses.txt`.

### Зауваження

Якщо у вас ще не встановлено Python або у вас немає інтерпретатора Python 3, ви можете завантажити його з за допомогою цієї команди в терміналі - brew install python@3.11

Скрипт запускається повністю локально і нікуди не надсилає ваші приватні ключі :)
